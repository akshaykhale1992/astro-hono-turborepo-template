# Turborepo Template with [Astro](https://astro.build/) and [Hono](https://hono.dev/).

## Why [Astro](https://astro.build/)?

In my opinion [Astro](https://astro.build/) provides the best Developer Experience if you want to build a front-end project. It support all the popular frameworks out there for building the UI components and their documentation is extensive if you get stuck during the development.

## Why [Hono](https://hono.dev/)?

HonoJs is a leading Typescirpt back-end framework that provides support for all the popular deployment options whether it's NodeJS, Bun, Deno, Cloudflare worker and many many other. The development is simple with "batteries included" for feature development, quick to build a full-stack application and yes, it supports JSX out of the box.

## System Requirements:

- pnpm version 11.5.0 or later.

## Usage:

1. pnpm install # to install the package dependencies
2. pnpm run dev # to start the Astro and Bun dev server.
3. Visit [http://localhost:4321/](http://localhost:4321/) for Astro Homepage and [http://localhost:3001/](http://localhost:3001/) for Hono Homepage.
